Backend For E-Commerce Telegram Bot With Mini APP

1. git clone https://github.com/azamat-9933/django-aio_3_tg_bot.git
2. CREATE DB FOR TG BOT - nasim_kutub in POSTGRESQL
3. CREATE VENV
4. ACTIVATE VENV
5. pip install -r requirements.txt
6. python manage.py makemigrations
7. python manage.py migrate
8. python manage.py createsuperuser
9. python manage.py runserver
10. OPEN http://127.0.0.1:8000/
11. LOGIN WITH SUPERUSER
12.
